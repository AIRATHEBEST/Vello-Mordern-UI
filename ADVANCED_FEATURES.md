## 🔧 Fix for Ollama Desktop App Connection

If your Ollama app is running but showing as **Disconnected**:

1. **Quit Ollama**: Right-click the tray icon and select "Quit".
2. **Set CORS Permissions**:
   - Open **Command Prompt (CMD)** as Administrator.
   - Run: `setx OLLAMA_ORIGINS "*"`
3. **Restart Ollama**: Open the app again.
4. **Pull a Model**: Run `ollama pull llama2` in CMD to ensure a model is available.
